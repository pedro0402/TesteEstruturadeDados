# Exercícios de Estruturas de Dados

## Descrição dos Exercícios

### 1. Pilha: Inversão de String
**Entrada:** `"Hello"`  
**Saída:** `"olleH"`  

### 2. Fila: Simulação
**Entrada:** `[]` (enfileirar `1`, `2`, desenfileirar)  
**Saída:** `1`  

### 3. Fila: Reversão
**Entrada:** `1, 2, 3`  
**Saída:** `3, 2, 1`  

### 4. Árvore: Crie uma estrutura de árvore binária

### 5. Árvore: Filhos
**Entrada:** `10 (filhos: 5, 15)`  
**Saída:** `[5, 15]`
